## Features

- ENS name resolution
- Portfolio value tracking
- Trader profiles
- TailwindCSS UI
- Keyboard shortcuts (`/` to focus search)
- TypeScript

## Development

```bash
# Install
pnpm install

# Dev server
pnpm dev

# Build
pnpm build

# Start
pnpm start
```

## Deployment

Deployed on Fly.io. Configuration in `fly.toml`.

```bash
# Deploy
fly launch
fly deploy
```

- React Router
